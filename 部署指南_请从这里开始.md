## 🔑 关键信息

### 数据库说明（重要！）

**你不需要安装 MySQL 或 PostgreSQL！**

本项目使用 **SQLite**：
- ✅ 文件型数据库，无需单独安装数据库服务器
- ✅ 运行 `npm run db:init` 会自动创建数据库文件
- ✅ 数据库文件位置：`prisma/dev.db`
- ✅ 零配置，开箱即用

### 服务器要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | 1核 | 2核 |
| 内存 | 1GB | 2GB |
| 存储 | 20GB | 40GB |
| 操作系统 | Ubuntu 20.04 | Ubuntu 22.04 |

### 需要安装的软件

部署指南会教你安装以下软件（都有详细步骤）：
1. ✅ Node.js 18
2. ✅ Git
3. ✅ PM2（进程管理器）
4. ✅ Nginx（Web 服务器）
5. ✅ （可选）Certbot（HTTPS 证书）

**不需要安装**：
- ❌ MySQL
- ❌ PostgreSQL
- ❌ MongoDB
- ❌ Redis

## ❓ 常见问题

### Q1: 我需要准备什么？
**A**: 
- 一台阿里云 ECS 服务器（Ubuntu 20.04/22.04）
- 服务器的 IP 地址和 SSH 密码
- （可选）一个域名

### Q2: 部署需要多长时间？
**A**: 
- 新手：30-60 分钟
- 有经验：15-30 分钟

### Q3: 我需要安装数据库吗？
**A**: 
不需要！项目使用 SQLite，运行 `npm run db:init` 会自动创建。

### Q4: 我的服务器是 CentOS，可以用吗？
**A**: 
可以，但需要将命令中的 `apt` 改为 `yum`。推荐使用 Ubuntu。

### Q5: 部署后如何更新代码？
**A**: 
使用项目中的 `deploy.sh` 脚本：
```bash
cd /var/www/life-moments/all2
./deploy.sh
```

### Q6: 遇到问题怎么办？
**A**: 
查看 [COMPLETE_DEPLOYMENT_GUIDE.md](./COMPLETE_DEPLOYMENT_GUIDE.md) 的第 20 章：故障排查
